### 3. 给 IDE Agent 的指令提示词 (Phase 4)

> **复制以下内容给 Agent：**
> 最后一个阶段是为系统创建一个 GUI。
> 1. **创建新文件 `src/gui_app.py**`：
> * 使用 Python 内置的 `tkinter` 库。
> * 导入 `src/main.py` 中的 `MetroPathPlanner` 类作为后端逻辑。
> 
> 
> 2. **实现界面逻辑**：
> * 创建一个主窗口类 `MetroGUI`。
> * **输入区**：两个 `Entry` 组件用于输入起点和终点。
> * **单选框**：使用 `Radiobutton` 选择策略（Shortest Path / Min Transfer）。
> * **输出区**：使用 `ScrolledText` 组件显示结果。
> * **按钮**：点击“查询”按钮时，获取输入框内容，调用 `planner.find_route`（需调整以适应新的输入方式），并将返回的字符串结果显示在输出区。
> 
> 
> 3. **集成**：
> * 确保 `MetroPathPlanner` 能够被 GUI 正确实例化和初始化（加载数据）。
> * 处理可能的异常（如找不到站点），并用 `messagebox.showerror` 弹窗提示用户。
