# 地铁换乘智能推荐系统

## 项目概述

本作业要求使用Python编程实现一个地铁换乘路径规划系统。该系统能够读取包含地铁线路、站点及换乘信息的CSV格式数据，并根据用户输入的起始站点和目的站点，智能生成最优换乘路线。

## 数据格式说明

系统所需的线路信息以CSV格式存储，包含以下字段：
- **站点ID**：唯一标识符，按升序排列
- **线路名**：所属地铁线路名称
- **站名**：站点名称
- **可换乘站点ID**：可换乘至其他线路的对应站点ID，多条换乘线路使用"/"分隔

同一线路的站点按照实际运行顺序排列，确保相邻站点具有前后顺序关系。

## 程序设计要求

### 数据结构设计
- 构建完整的地铁线路网络，每条线路应包含其所有站点信息
- 每个站点需记录其前后相邻站点（每个站点最多有一个前驱站点和一个后继站点）
- 建立换乘站点之间的关联关系

### 路径搜索算法
- 采用递归算法实现路径搜索功能
- 从起始站点开始，可向相邻的前后站点展开搜索
- 当搜索到换乘站点时，能够跳转至关联的其他线路继续搜索
- 为避免循环搜索和重复路径，需要标记已访问的站点

### 用户交互界面
- 提供清晰的输入提示，接收用户指定的起始站点和目的站点
- 按照指定格式输出完整的换乘方案
