# Requirements

Python程序设计第九周作业需求文档  

基础要求：  

- 掌握函数的定义、调用与参数传递
- 学会将一个较大问题分解为多个小函数，提高代码的模块化与可维护性
- 练习输入/输出处理与条件逻辑

编写一个薪资计算器，要求实现以下功能：  

- 输入员工的基本工资和奖金/津贴
- 计算应发工资（基本工资 + 奖金）
- 按分级税率计算个税（假设：应发工资 ≤ 免征点 免税，否则按照税率扣税，程序可以设定免征点和分级税率）
- 计算实发工资（应发工资 - 个税）
- 输出工资条（格式化显示员工姓名、应发工资、扣税金额、实发工资）

其他要求：  

必须记录与AI的完整对话过程  
说明哪些部分由AI生成，哪些由人工修改  
分析AI生成代码的优点和不足  
提交AI代码审查报告
