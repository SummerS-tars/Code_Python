# 如何快速学习函数文档

## 1. 快速学习函数文档的方法

可以用"先看签名 → 再看参数分类 → 最后对照示例"的套路：

### 1.1. 先看函数签名

```python
agg(func=None, ..., **kwargs) -> DataFrame
```

这告诉你：主要入口是 `func` 或 `**kwargs`（命名聚合）。

### 1.2. 参数区只抓"你会用到的类型"

`func` 支持：

- 单函数/函数名
- 列表（多函数）
- 字典（列 → 函数/函数列表）
- `None`（此时用 `**kwargs` 做命名聚合）

### 1.3. 命名聚合的关键句

"If func is None, **kwargs are used … output name is keyword"

这正对应你的写法：

```python
agg(Total_Price=('Total_Price','sum'))
```

### 1.4. 直接看与你最接近的例子

文档里 "named aggregation" 段落就是你当前用法的权威例子。

### 1.5. 一行总结

读文档时先定位主入口参数（这里是 `func`/**kwargs），再只读与自己用法匹配的那一小段说明和示例。
